Content Reporting System

The content reporting system allows users to flag inappropriate content for moderator review. When a user reports content, it appears in this queue for you to review and take appropriate action.

How to Handle Reports
  • Review the content - View the reported content to determine if it violates community guidelines
  • Dismiss the report - If the content doesn't violate guidelines, dismiss the report
  • Remove the content - If the content violates guidelines, remove it and possibly warn the user
All moderation actions are logged and can be reviewed by administrators.
